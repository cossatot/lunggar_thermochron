lunggar_thermochron
===================

Data and code for Styron et al. (in prep) manuscript for thermochronological modeling with Pecube, PiCloud and Python.


Transects are numbered 1-6 in the manuscript, but internally are referred to 
differently.

NLRT1: Transect 1
NLRT2: Transect 2
NLRT3: Transect 3
NLRT4: Transect 4
NMT:   Transect 5
CCTR:  Transect 6
